# Claw Chess

A lightweight browser chess game built as a static site for GitHub Pages.

## Features
- Legal move validation with `chess.js`
- Board flip
- Move log
- Captured pieces tracker
- Basic 10-minute clocks
- FEN copy

## Local run
Open `index.html` directly or run:

```bash
python3 -m http.server 8008
```

Then visit `http://127.0.0.1:8008`.
